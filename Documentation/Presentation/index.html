Projet

Application de location entre particuliers

concept:

  • Objets qui ne servent pas souvent
  • Objets qui ont de la valeur
  • Philosophie de ne pas acheter -> louer
  • Bricoleurs, vacanciers
  • Proximité / local
  • Point d'entée dans la crypto

Besoin utilisateurs

Besoins

  • Connection
  • Création d'un compte
  • Déposer (Supprimer / modifier) une annonce
  • Suivre une annonce
  • Gestion de litige
  • Décentralisation / sécurité / traçabilité

Mise en place

  • (connection) => Wallet Metamask
  • (compte) => Création d'une collection NFT & smart contract
  • (annonce) => Mint d'un NFT et création d'une location dans un smartcontrat
  • (suivi) => Smart contrat offre la possibilité de suivre ses annonces
  • (litige) => DAO pour la gestion de litige et gouvernance du projet
  • (décentralisation, ...) => Blockchain

Workflow utilisateur

  • Création d'une collection et location
  • Workflow complet location

Architecture

  • Un contrat basé sur ERC-4907 (et ERC-721)
  • Un contrat Factory et un contrat Collection
  • Contrat de location
